# disk_image_patch_071

Adds a short Pico-286 executable README next to the native app.

Copy this patch over the original SD-card filesystem root.  It adds:

- `MIPS_NATIVE/pico_286/README.md`

The README documents the main DOS controls, Fn shortcuts, on-screen keyboard,
disk image menu, and key preset editor controls for `pico_286`.
