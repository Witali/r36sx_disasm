Tiny MC
-------

This directory contains the Tiny MC native launcher.

Main executable:

  tiny_mc

The stock game transition script is now expected to launch it directly:

  /mnt/sdcard/MIPS_NATIVE/tiny_mc/tiny_mc /mnt/sdcard/MIPS_NATIVE

Tiny MC starts in /mnt/sdcard/MIPS_NATIVE so each standalone native program can
live in its own subdirectory.
